# Tazan
<hr>
KOSA I-ON 커뮤니케이션즈 채용확정형 교육 1기 <br>
1팀 "TAZAN" 김제민(🐵), 박수오(🐮), 윤현조(🐮), 임재혁(🐭)<br>
vue와 spring boot를 이용한 여행사 웹사이트입니다. <br>

주제 : 자유여행을 선호하는 20~30대를 타깃층으로 하여,
실제로 여행을 다녀온 사람들의 후기와 리뷰 그리고 여행계획을 공유하여
여행일정을 쉽게 작성할 수 있는 웹 사이트를 만들었습니다.
